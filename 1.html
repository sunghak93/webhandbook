태그(Tag)의 중요성

태그란 무엇인가?

  • 열리는 태그와 닫히는 태그 <~>,< /~>
  • 150개 이상의 태그가 존재 다 외울 필요는 없다.
  • 전세계 웹 페이지에서 쓰는 태그 통계=[어드밴스웹랭킹]
  • 보통 웹페이지 만드는데 태그 20~30개 사용 많아봐야 45개정도

최후의 문법 속성

속성(Attribute)이란?

  • 태그의 이름만으로 태그의 정보가 부족할 때 속성을 사용
  • 속성은 순서가 상관없음

이미지 삽입을 통해 속성을 알아보자

  • < img src="" width="">로 입력
    • 여기서 img는 이미지를 넣겠다는 태그이고 src는 이미지 파일이름을 나타내는 속성이고 width는 이미지의 크기를 나타내는 속성이다.
  • src=""사이에 사진 파일이름 + 확장자 쓰기, width=""사이에 크기를 지정하는 숫자나 퍼센트 쓰기
  • 직접넣어보자!!
  • 좋은 무료이미지 사이트=[언스플래쉬]

부모태그와 자식태그

  • < ul>와< li>태그는 반드시 부모자식간의 태그이다.
    • < li>는 목차(list)나타내는 태그이고 < ul>은 리스트들을 묶어주는 태그이다.
  • < ul>(Unordered List)대신< ol>(Ordered List)을 사용하면 자동으로 목차의 순서가 넘버링 된다.
  • < table>표 작성은 3대가 같이 다닌다. ex)< table>< /table>,< tr>< /tr>,< td>< /td>
  • < table border="숫자">라는 속성을 사용해야 표의 테두리가 나타남.

문서의 구조

  • 문서의 구조에는 본문을 설명하는< head>와 본문내용의< body>로 나뉘며 이 모든 것을 < html>과< /html>로 묶는다.
  • < head>~< /head>사이에는 < title> 제목 < /title>을 쓰고 < meta charset="">으로 본문의 저장방식을 설명해준다.
  • < body>~< /body>사이에는 본문의 내용을 작성한다.
  • 그리고 모든 페이지의 처음에는 관용적으로 < !doctype html>이라고 적어 html형식임을 알린다.

태그의 제왕 < a>태그 링크

  • < a>태그는 링크를 만든다.
  • < a href="주소">링크이름< /a>를 입력하면 된다
  • 새 창에 링크를 띄우고 싶을 때는< a href="주소" target="_blank">~< /a>로 입력하자.
  • 링크를 클릭하기전에 툴팁으로 알려주고 싶을땐< a href="주소" target="_blank" title="알리고자하는 내용">~< /a>를 입력하면 된다.
  • 구글을 링크해보자, 새 창에 구글링크

동영상 삽입하기 2가지 방법

  • 유튜브등의 영상을 퍼올 때< iframe width="너비" height="높이" src="동영상저장 주소" frameborder="숫자(테두리를말함)" allowfullscreen>< /iframe>
  • 자신의 피시의 영상을 올릴 때< video src="비디오파일주소+확장자" controls(컨트롤박스생성) loop(무한재생) muted(음소거) autoplay(자동재생) poster="썸네일 주소+확장>< /video>
